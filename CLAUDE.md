# Project: Five Senses

## Overview
F&B admin automation app for 5 Senses cafe (Singapore). Cat-themed, neobrutalist UI.

## Stack
- Single-file React (JSX via CDN)
- Lucide React icons
- Mock data in data.js
- No build tools — open in browser

## Style Direction
- Neobrutalism: thick black borders (3-4px), bold shadows, bright color blocks
- Palette: pink, yellow, teal, purple, cream backgrounds
- Cards with chunky rounded corners, bold sans-serif type
- Ref images: docs/style-ref/

## Current Phase
Playground first — nail the visual style before building features.
